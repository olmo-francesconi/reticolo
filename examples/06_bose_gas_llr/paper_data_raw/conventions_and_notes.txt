Francesconi, Holzmann, Lucini, Rago

ACTION (Eq. 2 in paper):
  S = sum_x [ (2d + m^2) phi_x* phi_x + lambda (phi_x* phi_x)^2
            - sum_{nu=1}^{4} ( phi_x* e^{-mu delta_{nu,4}} phi_{x+nu}
                               + phi_{x+nu}* e^{mu delta_{nu,4}} phi_x ) ]

  With phi_x = phi_{1,x} + i phi_{2,x}, split into real/imaginary:
  S = S^R + i sinh(mu) S^I

  S^R = sum_x [ (1/2)(2d+m^2) phi_{a,x}^2 + (lambda/4)(phi_{a,x}^2)^2
              - sum_{i=1}^{3} phi_{a,x} phi_{a,x+i}
              - cosh(mu) phi_{a,x} phi_{a,x+4} ]

  S^I = sum_x eps_{ab} phi_{a,x} phi_{b,x+4}   [antisymmetric tensor coupling]

  PARAMETERS USED IN ALL SIMULATIONS: lambda = m = 1.0

DEFINITION OF PHASE FACTOR:
  <e^{i phi}>_pq = Z / Z_pq
                = [integral rho(s) cos(mu s) ds] / [integral rho(s) ds]
                = [integral rho(s) cos(sinh(mu) * s) ds] / [integral rho(s) ds]
  Note: the argument is sinh(mu)*s, not mu*s, because the imaginary action
  is S = S^R + i sinh(mu) S^I, so the Fourier transform uses sinh(mu) as frequency.

DEFINITION OF OVERLAP FREE ENERGY:
  Delta_F = F - F_pq = -(1/V) log(<e^{i phi}>_pq)   [PER UNIT VOLUME]
  So: <e^{i phi}>_pq = exp(-V Delta_F)

CONVENTION MATCHING NOTES:

For a simulation with action parameters mass=1.0, lambda=1.0 and phase factor
defined as <cos S_I>_pq (cosine of the bare imaginary part):

The paper's phase factor is <e^{i phi}>_pq = <cos(sinh(mu) S^I)>_pq
where S^I = sum_x eps_{ab} phi_{a,x} phi_{b,x+4}.

If your simulation defines the phase factor directly as <cos(S_I)>_pq where
S_I is the same S^I field, then the comparison must account for the sinh(mu) prefactor:
  Paper: phase factor = <cos(sinh(mu) * S^I)>_pq
  Naive: <cos(S^I)>_pq

These are the same ONLY if your code uses sinh(mu) as the coupling coefficient in
the Fourier integral (i.e., integrates against cos(sinh(mu)*s), not cos(mu*s)).

The paper's action is the standard one where mu enters as an imaginary vector
potential in the time direction: the hopping term is phi_x* e^{-mu} phi_{x+4hat}.
This is the convention used broadly in the literature (Aarts, Gattringer).
The imaginary part of the action is S^I = sum_x eps_{ab} phi_{a,x} phi_{b,x+4},
which does NOT contain mu explicitly; mu only appears as sinh(mu) in the Fourier
transform of the partition function.

Match is straightforward if your simulation:
  (a) uses the same lattice action (hopping with e^{+/-mu} in time direction),
  (b) computes S^I = sum_x eps_{ab} phi_{a,x} phi_{b,x+4},
  (c) computes the overlap free energy as Delta_F = -(1/V) log(<cos(sinh(mu)*S^I)>_pq).

The reference values in the CSV files are Delta_F * 1e3 (i.e., divided by 1000).
